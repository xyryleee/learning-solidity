# Exercise: Token

Build a simple **ERC20-style token** on Avalanche C-Chain.

## Requirements

- Name and symbol (e.g. `MyToken`, `MTK`)
- Total supply set in the constructor
- `balanceOf(address)` — return token balance
- `transfer(to, amount)` — transfer tokens (emit event)
- Optional: `approve` / `transferFrom` (allowance)

## What to do

1. Implement your token in `Token.sol` (or use the stub and complete it).
2. Add unit tests in `test/` if you like.
3. Deploy to **Fuji testnet** and note the contract address.
4. In your Pull Request, mention your deployed contract address and Snowtrace link.

## Starter

See `Token.sol` in this folder. Replace the stub with your implementation.
